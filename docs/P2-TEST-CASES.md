# P2 测试用例

## 一、可自动化（单元测试）

### 1. 存档系统 (SaveSystem)
| 用例 | 预期结果 |
|------|----------|
| save() 后 hasSave() = true | ✅ |
| load() 返回保存的数据 | 各字段一致 |
| 无存档时 load() = null | ✅ |
| 无存档时 hasSave() = false | ✅ |
| deleteSave() 后 hasSave() = false | ✅ |
| save 的 version 字段存在 | ✅ |
| 世界变更差量存储正确 | changes 数组包含正确坐标和类型 |
| 物品栏存储/恢复正确 | 物品id和数量一致 |
| 存储空间数据正确保存/恢复 | storages 字段包含正确数据 |
| 损坏的 JSON 不崩溃 | load() 返回 null |

### 2. 雪人
| 用例 | 预期结果 |
|------|----------|

| 晴天计时结束后雪人消失 | blockType 变为 AIR |

### 3. 抱枕增强效果
| 用例 | 预期结果 |

| 沙发旁有抱枕 → 减缓 30% | rate = 0.3 |
| 检查1格范围内是否有抱枕 | hasPillow() 正确判断 |
